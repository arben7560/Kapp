# K-App

## Vision

K-App is a premium Korean learning application built with React Native and Expo.

The objective is not simply to teach Korean vocabulary, but to recreate the experience of living in Seoul through immersive, realistic and interactive situations.

The application targets beginners up to intermediate learners.

The user should feel guided like in a modern Apple-quality application.

---

# Main Philosophy

Learning should happen through context.

The user should:

- understand
- listen
- speak
- interact
- progressively become autonomous

The application prioritizes immersion over memorization.

---

# Main Modules

## Hangul

Learning:

- vowels
- consonants
- tense consonants
- aspirated consonants
- batchim

Exercises gradually increase in difficulty.

---

## Vocabulary

Vocabulary is organized by themes.

Examples:

- First Steps
- Food
- Transport
- Business
- K-Drama
- Romance
- Nightlife

Some modules are free.

Some require Premium.

---

## Listening

Audio-based exercises.

Objectives:

- listening comprehension
- vocabulary recognition
- sentence understanding

---

## Speaking

Real conversation simulations.

Examples:

- Café
- Restaurant
- Metro
- Airport

The user speaks with AI.

The application evaluates the answer.

---

## AI Teacher

Conversational Korean teacher.

Capabilities:

- conversation
- grammar
- pronunciation
- correction
- translation

---

## Immersion

The core feature of the application.

Each immersion reproduces a realistic Korean situation.

Examples:

- ordering coffee
- restaurant
- taking the subway
- airport

Immersions should feel realistic and natural.

---

# Design Philosophy

The application follows an Apple-inspired design language.

Objectives:

- premium
- elegant
- minimal
- modern
- consistent

The interface should never feel cluttered.

Every screen should belong to the same design system.

---

# Typography

Fonts:

- Outfit
- Noto Sans KR

Typography should remain consistent across the entire application.

---

# Colors

Dark premium interface.

Glassmorphism is used carefully.

Avoid excessive gradients or flashy colors.

Accent colors are used only when meaningful.

---

# Technical Stack

Framework:

- React Native
- Expo

Routing:

- Expo Router

State:

- Zustand

Storage:

- AsyncStorage

Language:

- TypeScript

---

# Code Principles

Prefer:

- reusable components
- centralized styles
- modular architecture
- responsive layouts

Avoid:

- duplicated code
- unnecessary refactoring
- hardcoded values
- visual inconsistencies

---

# Responsive Support

The application must work correctly on:

- small phones
- large phones
- tablets

Layouts should adapt naturally.

---

# Performance

Keep animations smooth.

Avoid unnecessary renders.

Optimize heavy components.

---

# Accessibility

Respect readable font sizes.

Maintain adequate touch targets.

Keep sufficient color contrast.

---

# Premium

Premium unlocks advanced immersion content.

The free version must still provide a satisfying learning experience.

Premium should feel like an expansion rather than a restriction.

---

# Development Goal

Every improvement should move K-App closer to a production-ready application suitable for publication on the App Store and Google Play.

The application should feel cohesive, polished and professionally designed.
