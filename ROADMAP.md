# K-App Development Roadmap

_Last updated: July 2026_

---

# Vision

K-App aims to become the reference mobile application for French speakers learning Korean through realistic immersion.

The objective is to deliver an experience that feels closer to a premium Apple application than to a traditional language-learning app.

# Long-Term Goal

K-App should become one of the highest-quality Korean learning applications available, combining:

- immersive learning
- realistic conversations
- premium interface
- excellent performance
- long-term user retention
- continuous content expansion
- Apple-level user experience

Every new feature should reinforce these principles without compromising simplicity, consistency or learning quality.
